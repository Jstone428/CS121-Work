lawOfCosines <- function(a, b, theta) {
    return(sqrt(a^2 + b^2 - 2 * a * b * cos(theta)))

Graphics

Write a function canvas() that takes a vector of two numbers to draw a blank canvas on that scale. 

canvas <- function(x, y) {
    return(plot(x:y, type = "n", xlim = c(1, 100), ylim = c(1, 100), xlab = "", 
        ylab = "", asp = 1))
}
canvas(6, 8)
